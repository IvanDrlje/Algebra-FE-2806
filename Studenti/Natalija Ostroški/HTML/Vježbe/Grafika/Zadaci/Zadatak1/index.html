Rad s grafikom u HTML-u

Format grafičkih datoteka

Za prikaz grafike na webu koriste se sljedeći formati:

JPEG
Joint Photographic Experts Group
Koristi se za prikaz fotografija
GIF
Graphics Interchanger Format
Koristi se za prikaz jednostavne grafike (crteža, ilustracije i sl.)
PNG
Portable Network Graphics
Koristi se za prikaz fotografija i crteža uz mogućnost primjene transparencije
SVG
Scalable Vector Graphics
Koristi se za prikaz dvodimenzionalnih vektorskih slika (ikone, logotip i sl.)
