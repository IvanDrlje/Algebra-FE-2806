Uvod u HTML

HTML je jezik za obilježavanje koji koristimo za izradu web stanica.
Njime definiramo strukturu i sadržaj web stranica.

Povijest HTML-a

HTML je nastao početkom 1990-tih godina.

Uređivanje teksta

Tekst je najčešći sadržaj web stranice.
Da bi dobio odgovarajuću ulogu na stranici i bio prikazan na određen način uređujemo ga HTML-om.

HTML atributi

Unutar početne oznake elemenata, nakon naziva elemenata po potrebi pišemo i atribute.
Atributi internetskim preglednicima daju dodatne informacije o elementu. O samom atributu ovisi o kojim
informacijama je riječ.
Atribut se sastoji od imena i vrijednosti atributa.
