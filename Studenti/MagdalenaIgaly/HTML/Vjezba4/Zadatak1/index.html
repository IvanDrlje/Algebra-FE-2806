Formati grafickih datoteka

Za prikaz grafike na webu, koriste se sljedeci formati

JPEG
Joint Photographic Experts Group
Koristi se za prikaz fotografija
GIF
Graphic Interchange Format
Koristi se za prikaz jednostavne grafike
PNG
Portable Network Graphic
Koristi se za prikaz fotografija i crteza uz mogucnost primjene transparencije
SVG
Scalable Vector Graphic
Koristi se za prikaz 2D vektorskih slika
